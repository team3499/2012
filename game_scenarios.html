Game Scenarios

Hybrid Phase Shooting

  1. There is a ball in the shooter.
  2. The robot is aimed so that the entire bottom rectangle is in the camera's view.
  3. The robot performs the autonomous shooting operation.
  4. If there is another ball in the magazine, the robot loads it into the shooter.

Loading a Ball into the Shooter

  1. There is a ball in the shooter.
  2. The operator drives the robot to a place in range of the baskets and aims the shooter at the targets well enough that the entire bottom rectangle is in the camera's view.
  3. The operator presses a button, for one of the baskets, on the shooter joystick commanding the robot to aim and shoot.
  4. The robot performs its autonomous shooting operation.

Remote Operation Phase Shooting

Autonomous Shooting

  1. The robot is commanded to shoot at a given basket.
  2. There is a ball in the shooter.
  3. The entire bottom target rectangle is in the camera's view.
  4. An image is captured from the camera and processed to locate the bottom rectangle within the image.
  5. The distance to the backstop is determined from the rangefinder.
  6. Using the target location and distance, the robot turns the calculated angle required to aim the shooter at the basket, during which, the gyro provides feeback for the controlled turning.
  7. The distance to the backstop is measured again.
  8. Using the distance to the backstop and the height of the target basket, the robot moves the shooter to the calculated elevation, during which the accelerometer provides feedback for the controlled arm movement, then shoots the ball by running the shooter's motors at the calculated speed and duration.

Operator Shooting

  1. There is a ball in the shooter.
  2. The operator drives the robot to some distance from the target, aims the robot at the target, and adjusts the shooter's elevation with the shooter joystick.
  3. The operator commands the robot to shoot at one of a few pre-selected speeds by pressing the associated button on the joystick.
  4. The robot shoots the ball by running the shooter's motors at the pre-selected speed and duration.
